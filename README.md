# Ferma Prime Test

## Описание
Программа для проверки простоты чисел с использованием теста Ферма.

## Возможности
- Проверка чисел на простоту с помощью теста Ферма
- 100 итераций теста для высокой точности
- Поддержка ввода числа через аргумент командной строки или интерактивный режим
